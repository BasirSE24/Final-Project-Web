##  Features

-  **Form Validation** – Ensures proper data entry before submission  
-  **Accordion Sections** – Expand and collapse content neatly  
-  **Day/Night Theme Toggle** – Switch between color themes  
-  **Dynamic Date & Time Display** – Real-time updates  
-  **Popups and Toast Messages** – User-friendly notifications  
-  **Club Information Cards** – Show player or team details with images  
-  **Responsive Design** – Works on mobile and desktop

##  How to Run

1. Clone or download this project folder.  
2. Open it in **Visual Studio Code**.  
3. Run the `index.html` file in your browser:
   - Right-click `index.html` → **Open with Live Server** *(recommended)*  
   - or simply double-click `index.html`.
